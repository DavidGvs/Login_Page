Login page

Permite o registo de um Utilizador numa base de dados, permitindo o acesso à sua conta através de uma página de Login e desta forma ser redirecionado a uma página de gestão de conta onde será permitido a atualização dos dados pessoais e de apagar por completo a sua conta.

Download Download View Login page on GitHub

Página Inicial

  • Aqui o utilizador consegue fazer login na sua conta ou então registar-se caso ainda não o tenha feito.

Página de Registo

  • Nesta página o utilizador consegue inserir os seus dados de forma a ficar registado na base de dados e desta forma ser redirecionado para a página inicial, onde poderá fazer Login e aceder à página de gestão de conta.

Página de Gestão de Conta

  • O utilizador conseguirá, através desta página, gerir a sua conta, atualizando os seus dados ou então apagar por completo a sua conta da base de dados.
